### 🚀 How to Use

1. **Visit the website:** https://PimplesOnNose.github.io/journey-to-the-west/
2. **Navigate pages:** Use arrow buttons or keyboard arrows
3. **Switch language:** Click "English" or "中文" button
4. **Play audio:** Click the play button
5. **Auto-play:** Click the "Auto" button for continuous reading

### 📱 Controls

- **←/→ Arrow Keys:** Previous/Next page
- **Spacebar:** Play/Pause audio
- **Enter:** Toggle auto-play
- **Language Button:** Switch between English/Chinese

### 🔧 Local Development

To run locally:
```bash
cd journey-to-the-west
./start.sh
```

Or use Python:
```bash
python3 -m http.server 8000
```

Then open http://localhost:8000
